Для Sequelize:
-все таблицы названы следующим образом:
--[сущность].[entity].[ts]

Для GraphQL:
-все перечисления (enum) названы следующим образом:
--[сущность].[entity].[ts]


-все DTO названы следующим образом:
--[название input-а из каталога type].[dto].[ts]

-все type называем следующим образом:
--[название сущности].[type].[ts] - для описание типов
--[тип данных/сущность/метод].[input].[ts] - для описания инпутов


Если название состоит из нескольких слов разделяем знаком "-", например:
sign-up.dto.ts
sing-in.input.ts
и т.д.
